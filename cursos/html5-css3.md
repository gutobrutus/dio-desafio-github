## Estrutura básica de um HTML
```html
<!DOCTYPE html>
<html>
    <head>
        <meta>
        <title></title>
    </head>
    <body>
    
    </body>
</html>
```
Por ser uma linguagem de marcação de hipertextos, essa marcação é realizada através de tags (etiquetas). A descrição das tags é facilmente encontrada na web.
